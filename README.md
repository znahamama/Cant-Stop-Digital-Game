# 🎲 Can't Stop – Java Digital Board Game

A Java-based desktop version of the classic board game **Can't Stop**, featuring 2–4 player support, AI opponents, save/load functionality, and accessibility enhancements for colorblind users.

---

## 🚀 Features

- 🎮 **2, 3, or 4 Players**: Add or remove players in setup; play with both humans and AI.
- 🧠 **Computer Players**:
  - **Easy Mode**: Randomized choices
  - **Hard Mode**: AI makes strategic moves based on runner progress
- 🎲 **Core Gameplay**:
  - Roll dice, select up to two combinations
  - Roll again or end your turn
  - Column ownership and valid runner rules enforced
- 💾 **Save/Load System**:
  - Save your game from the options menu
  - Load it anytime from the main menu
- ♿ **Accessibility**:
  - Colorblind mode uses numeric indicators for player markers

---

## 📦 How to Run

1. **Download or clone the repository**
2. Navigate to the project directory
3. Run the executable JAR file:

```bash
java -jar "Can'tStop.jar"
```
